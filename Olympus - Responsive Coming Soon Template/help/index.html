B) Font Used

Open Sans - https://fonts.google.com/specimen/Open+Sans
Font Awesome - https://fontawesome.com/

You can change font from CSS file.

C) Color Schema

Olympus comes with 6 predefined colors green, blue, orange, purple, yellow and grey.

You can change the color of template by the following code in demo HTML sources, <head> tag:

<link class="colors" rel="stylesheet" href="layout/colors/green.css" type="text/css">

D) Countdown Date

To change the date of the countdown, go to layout/js/ folder and open main.js file. Find to:

austDay = new Date("December 25, 2023 00:00:00");

and change to the date you want.

E) Background Image - Video

Single Image

To change background, open layout/style.css file go to line 221, change url (../images/image/bg.jpg) with your image url.

Image Slideshow

In order to add your own photos to the background slider, there are a few steps to make:

  1. You need to create the slideshow pictures. In the demo, the pictures are at least 1024x785px in dimensions, representing the biggest most-common screen resolution. You can make them as big as you want, but is important to make them at least 1024x785px to fit almost all the screens. Save the pictures inside the images folder.

  2. Go to slide.html and add your image with the following structure.

    <li class="slide">
    <img src="images/slide/1.jpg" alt="" />
</li>

Youtube Background

In order to add your own photos to the background slider, there are a few steps to make:

  1. Open layout/js/main.js file and go to line 89:

    videoId:'HIHQWS84DDo',

  2. Change the Youtube video id HIHQWS84DDo with your video id:

    https://www.youtube.com/watch?v=HIHQWS84DDo

F) Subscribe Form

You need to configure subscribe.php to get e-mails from the users.

Change those variables on line 11 with the real name & e-mail address to send the mail information:

$toName = "Olympus";
$toAddress = "email@sitename.com";

G) Google Maps

Please, get an API key here:
https://developers.google.com/maps/documentation/javascript/get-api-key

To change Google Maps options, you need to change variables in demo pages:

var arrMap = {
	"title":"Olympus Main Office",
	"latitude":37.800976,
	"longitude":-122.428502,
	"zoom":12,
	"marker":"layout/images/map-marker-green.png",
	"color":"#0fe0ba"
}
